# Unityでつくる

## ターゲットのオブジェクトを作成

1.	空のゲームオブジェクトを作成
ヒエラルキーに空のゲームオブジェクトを作成し、名前を Target に設定します。

2.	子オブジェクトを追加
Target の子オブジェクトとして、Sphere と Capsule を追加します。

![](https://storage.googleapis.com/zenn-user-upload/37f364675b5d-20241204.gif)

3.	形状の調整
子オブジェクトの位置やスケールを調整して、人型のような形状に仕上げます。形は自由なので、お好みでアレンジしてください！

![](https://storage.googleapis.com/zenn-user-upload/babe2e6df653-20241204.png)

4.	プレハブ化
ヒエラルキーで作成した Target をプロジェクトビューにドラッグ&ドロップしてプレハブとして保存します。これにより、シーン内で何度でも簡単に再利用できます。

![](https://storage.googleapis.com/zenn-user-upload/bb332d67f894-20241204.gif)

5.	コンポーネントの追加
Target に Rigidbody と適切な Collider コンポーネント（例: Sphere Collider, Capsule Collider）をアタッチします。

>Rigidbody と Collider コンポーネントを追加することで、ターゲットに物理挙動と当たり判定を設定します。これにより、以下のような挙動が可能になります：
>1.	射撃の検知
Rigidbody によって、外部からの物理的な力や衝突がプログラム側で検知されます。例えば、射撃時の衝突イベントをトリガーにしてスコア加算やエフェクトの発生などを実装できます。
>2.	地面や他のオブジェクトとの当たり判定
Collider を設定することで、ターゲットは地面やプレイヤーをすり抜けることなく、現実的な衝突判定を行います。これにより、ターゲットが物理的に自然な挙動を示します。

![](https://storage.googleapis.com/zenn-user-upload/81e733d48fcf-20241207.gif)

:::message
**注意点**
- igidbody の Use Gravity をオンにすると、ターゲットが自然に重力の影響を受けます。
- Collider の形状はオブジェクトの形に合わせて適切なものを選択してください。例えば、Sphere Collider は球状部分、Capsule Collider は胴体部分に適しています。
- Is Trigger をオンにすると、衝突は検知されますが、物理的な力が適用されなくなるため、用途に応じて設定を切り替えましょう。
:::

## Targetスクリプトの作成
```csharp:Target
using UnityEngine;
using System.Collections;

public class Target : MonoBehaviour
{
    public int scoreValue = 10;                 // 的の得点
    public float speed = 2.0f;                  // 移動速度
    public float moveRange = 3.0f;              // 移動範囲
    public GameObject explosionEffectPrefab;    // 爆破エフェクトのプレハブ
    private Vector3 startPosition;              // スタート位置
    private bool movingRight = true;            // 右に動いているかどうか
    private float leftBoundary;                 // 左端の位置
    private float rightBoundary;                // 右端の位置

    void Start()
    {
        // スタート位置を取得
        startPosition = transform.position;

         // 左端と右端を初期位置と移動範囲から計算
        leftBoundary = startPosition.x - moveRange;
        rightBoundary = startPosition.x + moveRange;
    }

    void Update()
    {
        MoveTarget();
    }

    private void MoveTarget()
    {
        if (movingRight)
        {
            // 右方向に移動
            transform.position += Vector3.right * speed * Time.deltaTime;

            // 右端に到達したら反転
            if (transform.position.x >= rightBoundary)
            {
                movingRight = false;
            }
        }
        else
        {
            // 左方向に移動
            transform.position += Vector3.left * speed * Time.deltaTime;

            // 左端に到達したら反転
            if (transform.position.x <= leftBoundary)
            {
                movingRight = true;
            }
        }
    }

    public void Hit()
    {
        // 爆破エフェクトを生成
        Instantiate(explosionEffectPrefab, transform.position, Quaternion.identity);

        // スコアを追加
        // GameManager.Instance.AddScore(scoreValue);

        // ゲームオブジェクト(ターゲット)を非表示
        gameObject.SetActive(false);
    }
}
```

# Unityを理解する

## Unityでターゲットに必要なことをまとめる

本講座では、左右に移動するターゲットを射撃し、ポイントを獲得するFPSゲームを作成します。さらに、ヒット時に爆発エフェクトを追加し、ゲームの演出を強化します。

ターゲット（Target）には以下の機能が必要です。
•	移動：左右に往復する動き
•	爆発：ヒット時に爆発エフェクトを表示
•	ポイント計算：ヒット時にGameManagerスクリプトを通じてスコアを計算

これらの機能をTargetスクリプトで実装していきます。

## Targetスクリプトの記述

Targetの挙動を実装するためのスクリプトを作成します。

### 往復移動

#### スタート関数内の処理

ターゲットの開始位置（Transformのワールド座標）と、移動範囲を表す変数を基に、ターゲットが移動できる右端と左端のTransform（ワールド座標）を計算します。

#### アップデート関数内の処理

ターゲットを右方向に移動させ、if文を用いて右端のTransformに到達した際に、移動方向を左方向に切り替えます。

続いて、if文で左端のTransformに到達した際には、移動方向を右方向に切り替えます。

これを繰り返すことで、ターゲットの往復移動を実現します。

#### 変数の設定

移動範囲や移動速度については、外部から変更できるようにpublic変数として宣言します。これにより、柔軟な設定が可能になります。
