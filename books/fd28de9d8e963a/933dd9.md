# 課題: プレイヤー操作を拡張しよう！

以下の2つの課題から好きなものを1つ選び、実装に挑戦してみましょう！

# 課題 A: 走り操作を実装しよう！（難易度: ★☆☆）

## 目的:

Shiftキーを押しながら移動すると、移動速度が2倍になる機能を実装します。

## 手順:

Shiftキーの入力を検知します。
Shiftキーが押されている間、移動速度を通常の2倍に設定します。
Shiftキーを離したら、通常の速度に戻します。

:::details ヒント : サンプルコードを見る
```csharp
using UnityEngine;

public class PlayerController : MonoBehaviour
{
    public float normalSpeed = 5f; // 通常時の速度
    public float runSpeed = 10f;  // 走る時の速度
    private bool isRunning = false;

    public void OnRun(InputAction.CallbackContext context)
    {
        isRunning = context.performed; // Shiftキーの押下状態を検知
    }

    private void HandlePlayerMovement(Vector2 moveInput)
    {
        // 速度を判定
        float currentSpeed = isRunning ? runSpeed : normalSpeed;

        // 移動方向を計算
        Vector3 direction = transform.forward * moveInput.y * currentSpeed
                            + transform.right * moveInput.x * currentSpeed;

        // 現在の位置を更新
        transform.position += direction * Time.deltaTime;
    }
}

```
:::

## 課題 B: ジャンプ機能を実装しよう！（難易度: ★★☆）

## 目的:

Spaceキーを押すとプレイヤーがジャンプする動作を実装します。
ジャンプ後は重力の影響を受けて自然に落下します。

## 手順:

Spaceキーの入力を検知します。
プレイヤーが地面にいる場合のみ、上方向に力を与えてジャンプさせます。
地面との接触を判定する方法を実装します。

:::details ヒント : サンプルコードを見る
```csharp
using UnityEngine;

public class PlayerController : MonoBehaviour
{
    public float jumpForce = 5f; // ジャンプの力
    private Rigidbody rb;

    void Start()
    {
        rb = GetComponent<Rigidbody>();
    }

    public void OnJump(InputAction.CallbackContext context)
    {
        if (context.performed && IsGrounded())
        {
            rb.AddForce(Vector3.up * jumpForce, ForceMode.Impulse); // 上方向に力を加える
        }
    }

    private bool IsGrounded()
    {
        // プレイヤーが地面に接触しているかを判定
        return Physics.Raycast(transform.position, Vector3.down, 1.1f);
    }
}
```
