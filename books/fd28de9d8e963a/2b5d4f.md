# Unityの準備をしよう：基本操作を徹底解説

## シーンビューの基本操作：直感的な編集を始めよう

シーンビューでゲームオブジェクトの配置や編集が可能です。以下に基本的な操作方法を紹介しますので、今後の操作の参考にしてみてください。

| 操作 | 説明 |
|----------------------|---------------------------------------------------------------|
| **ゲームオブジェクトの選択** | シーンビューまたはヒエラルキービューでオブジェクトを`クリック`して選択 |
| **視点の移動** | `右クリック`しながら`マウス`を動かす。また、`W`、`A`、`S`、`D`で移動 |
| **ズームイン/アウト** | `マウスホイール`を前後に回転 |
| **ゲームオブジェクトの移動** | `W`を押して移動ツールを選択 |
| **ゲームオブジェクトの回転** | `E`を押して回転ツールを選択 |
| **ゲームオブジェクトのスケーリング** | `R`を押してスケーリングツールを選択 |
| **フォーカスの調整** | `F`を押して選択中のゲームオブジェクトにカメラを合わせる |

:::details その他のUnity ショートカットキー
| **ショートカットキー (Windows)** | **ショートカットキー (Mac)** | **説明**                         |
|----------------------------------|------------------------------|----------------------------------|
| `Ctrl + D`                       | `Cmd + D`                    | 選択したゲームオブジェクトを複製       |
| `Delete`                         | `Delete`                     | 選択したゲームオブジェクトを削除       |
| `W`                               | `W`                          | ゲームオブジェクトの移動ツール         |
| `E`                               | `E`                          | ゲームオブジェクトの回転ツール         |
| `R`                               | `R`                          | ゲームオブジェクトのスケール（拡大縮小）ツール |
| `F`                               | `F`                          | 選択したゲームオブジェクトにフォーカス |
| `Ctrl + Z`                        | `Cmd + Z`                    | 操作の取り消し (Undo)           |
| `Ctrl + Y`                        | `Cmd + Shift + Z`            | 操作のやり直し (Redo)           |
| `Alt + 左ドラッグ`                | `Option + 左ドラッグ`        | カメラを回転させて視点を移動     |

| `Space` (ツールバーで)            | `Space` (ツールバーで)       | 現在のツールを最大化             |
| `Shift + F`                       | `Shift + F`                  | 選択したゲームオブジェクトにカメラを追従 |

# Unityを学ぶ第一歩：基礎からのステップバイステップ

この章では、Unityの基本操作を実際に手を動かしながら学んでいきます。もし概念や用語が難しいと感じた場合は、前の記事を参考にして復習してみてください。

## ゲームの基本単位を作成しよう：ゲームオブジェクトの生成

**1. Unityエディタを開く**
- Unityエディタを開き、ヒエラルキービューにアクセスします。

**2. オブジェクトの生成**
- ヒエラルキービュー内で右クリックし、「3D Object」 -> 「Capsule」を選択します。
- カプセルオブジェクトがシーンに生成されます。

![](https://storage.googleapis.com/zenn-user-upload/b211a3d7fa5c-20241129.gif)

:::message
オブジェクトを生成するとデフォルトで (0, 0, 0) に配置されるため、位置の調整を忘れないこと。
:::

## 	時短テクニック：ゲームオブジェクトを簡単に複製する方法

**1. 複製するオブジェクトの選択**
- 複製したいオブジェクトをヒエラルキービューで選択します。

**2. 複製の実行**
- Windowsでは `Ctrl + D`、Macでは `Cmd + D` を押して、選択したオブジェクトを複製します。

![](https://storage.googleapis.com/zenn-user-upload/d3d163fd1169-20241129.gif)

:::message
複製後の名前は自動的に「オブジェクト名(Clone)」になるため、わかりやすい名前に変更するのが推奨です。
:::


## 間違えても安心！オブジェクトの削除と取り消し方法

**1. 削除するオブジェクトの選択**
- 削除したいカプセルオブジェクトをヒエラルキービューで選択します。

**2. 削除の実行**
- `Delete` キーを押すか、右クリックして「Delete」を選択します。
- **注意**: 誤って削除してしまった場合は、 `Ctrl + Z` （Macでは `Cmd + Z`）で取り消しができます。

![](https://storage.googleapis.com/zenn-user-upload/11ee49989a9b-20241129.gif)


## 複数オブジェクトの操作を効率化！範囲選択と複数選択のコツ

**1. 範囲選択**
- `Shift` キーを押しながら、ヒエラルキービューで最初と最後のゲームオブジェクトをクリックして範囲選択を行います。

![](https://storage.googleapis.com/zenn-user-upload/0485511e2ae6-20241129.gif)


**2. 複数選択**
- `Ctrl` キー（Macでは `Cmd`）を押しながら、選択したい複数のゲームオブジェクトをクリックして選択します。
- **ポイント**: 複数のゲームオブジェクトをまとめて動かしたり、削除したいときに便利です。

![](https://storage.googleapis.com/zenn-user-upload/f505cf18f824-20241129.gif)


## オブジェクトを自由自在に操作しよう！移動・回転・スケールの基礎

**1. 操作するゲームオブジェクトの選択**
- ヒエラルキービューでカプセルオブジェクトを選択します。

**2. 操作ツールの選択**
- ツールバーから「移動（Move）」、「回転（Rotate）」、「スケール（Scale）」ツールを選びます。

**3. シーンビューで調整**
- シーンビューに表示されるハンドルをドラッグして、ゲームオブジェクトの位置、回転、サイズを調整します。
- **ショートカット**:
  - `W`: 移動ツール
  - `E`: 回転ツール
  - `R`: スケールツール

:::message
ハンドル操作が難しい場合は、インスペクターで数値を直接入力して調整するのも有効です。
:::

![](https://storage.googleapis.com/zenn-user-upload/512dbf73e343-20241129.gif)

# Unityで初めての作品を作ろう！カプセルとカメラで簡単3Dゲーム

`Capsule`と`Camera`を使って、3Dゲームでのプレイヤー視点を簡単に作成してみましょう。

**1. カプセルオブジェクトの作成**
- ヒエラルキービューでカプセルオブジェクトを作成します。

**2. 名前の決定**
- 「PLAYER」などの新しい名前を入力します。

**3. カメラオブジェクトの選択**
- ヒエラルキービューでカメラオブジェクトを選択します。

**4. 親子関係の設定**
- カメラオブジェクトをドラッグして「PLAYER」オブジェクトの上に移動させます。
- カメラが「PLAYER」オブジェクトの子オブジェクトとして設定され、プレイヤーの動きに追従するようになります。

:::message
親子関係の設定で、子オブジェクトの位置が親オブジェクト基準になることを意識する必要があります。
:::

**5.プレハブ化をする**

- 作成したゲームオブジェクトをプロジェクトパネルのプレハブ用フォルダにドラッグ＆ドロップします。
- これでゲームオブジェクトがプレハブとして保存され、いつでも再利用可能になります。

![](https://storage.googleapis.com/zenn-user-upload/7292cd089839-20241201.gif)

:::message
プレハブのインスタンスを直接編集しても、元データには反映されないことに注意。
:::
