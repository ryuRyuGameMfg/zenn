## 問題 6: グローバル変数とローカル変数
次のコードについて、以下の質問に答えてください。

```csharp
public class Example : MonoBehaviour
{
    public int globalVariable = 100;

    void Start()
    {
        int localVariable = 50;
        Debug.Log(globalVariable);
        Debug.Log(localVariable);
    }

    void Update()
    {
        Debug.Log(globalVariable);
        // Debug.Log(localVariable); // この行はエラーになる
    }
}
```

1.`globalVariable`にアクセスできる関数はどれですか？
(a) Start()
(b) Update()
(c) 両方

2.`localVariable`にアクセスできる関数はどれですか？
(a) Start()
(b) Update()
(c) 両方
