# 自由にテーマを決めてシーンを構築

- 自然の風景、公園、未来都市など、好きなテーマを選んでシーンを構築しましょう。
- 3Dオブジェクトを組み合わせて、より複雑な構造物を作成してみてください。

> **後ほど3Dオブジェクトのインポート方法についても詳しく説明しますが、Unityアセットストアを利用することで、木や川、町、家など、さまざまな3Dオブジェクトを簡単に導入することができます。**

https://assetstore.unity.com/

## シーンを構築するヒント！

### 1. ライトオブジェクトを使用する
- **Directional Light**: 太陽光をシミュレートし、全体の明るさを調整。
- **Point Light**: 特定のポイントに光を配置し、ランプや照明効果を表現。
- **Spot Light**: 円錐状の光を使用して、スポットライトや懐中電灯を再現。

### 2. リジッドボディを使った物理的な動きの実装
- **Rigidbody コンポーネント**を追加して、物理挙動をシミュレート。
  - 重力を適用し、オブジェクトが自然に落下する動きを確認。
  - **Mass（質量）** を調整して、重量感の違いを体験。
  - 衝突が不要な場合は「Is Trigger」をオンにして、物理的な衝突を無効化し、イベントだけを発生させる。

### 3. カラーやマテリアルの作成と設定
- プロジェクトビューでマテリアルを作成し、オブジェクトに適用。
  - **手順**:
    1. プロジェクトビューで右クリック → `Create` → `Material` を選択。
    2. 名前を「GroundMaterial」などに変更。
    3. インスペクターで「Albedo」の色を設定。
    4. 必要に応じて「メタリック」や「スムースネス」のスライダーを調整し、質感を変更。
- 作成したマテリアルをドラッグ＆ドロップしてオブジェクトに適用。

### 4. パーティクルの追加
- **Particle System** を使用して、シーンに動きを追加。
  - 雨、火花、煙などのエフェクトを作成して、シーンをより魅力的に演出。

# シーンを保存し、共有

1. 完成したシーンを保存します。
2. Unityの画面をスクリーンショットで撮影。
3. **どこでもUnity教室のDiscordサーバー**にシーンを共有しましょう！

[どこでもUnity教室 Discord サーバー](https://discord.gg/AhxnbR4D)

### シーンの保存方法

1. **シーンを保存する準備**  
   - シーンビューで作成したシーンを確認します。
   - 保存したい内容が正しいかチェック。

2. **保存手順**  
   - メニューバーから `File` → `Save` をクリックします。
   - または、`Ctrl + S`（Macの場合は `Cmd + S`）のショートカットキーを押します。

3. **シーンの名前と保存場所を指定**  
   - 初めて保存する場合は、ポップアップウィンドウが表示されます。
   - プロジェクト内の適切なフォルダ（例: `Assets/Scenes`）を選択し、シーン名を入力します。
   - 「Save」ボタンをクリックして保存を完了します。

4. **保存したシーンの確認**  
   - プロジェクトビューで保存したシーンファイル（拡張子 `.unity`）を確認します。

> **ヒント**
> シーンを保存するときは、進行中のプロジェクトが失われないように、定期的に保存する習慣をつけましょう！
