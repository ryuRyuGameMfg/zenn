# 問題 1: スクリプトを作成する  

**以下の手順で新しいスクリプト `PikachuController` を作成してください。**  

1. Unityのプロジェクトビューで `+` をクリックし、`MonoBehaviour Script` を選択します。  
2. スクリプトの名前を `PikachuController` とします。  
3. 作成したスクリプトをダブルクリックして、エディタで開いてください。

# 問題 2: スクリプトを記述する  

**作成した `PikachuController` に以下の機能を追加してください。**

1. ゲーム開始時に「ピカチュウが登場しました！」とログを表示する。  
2. 毎フレーム「ピカチュウが動いています」とログを表示する。

:::details ヒント : サンプルコードを見る
```csharp
using UnityEngine;

public class PikachuController : MonoBehaviour
{
    void Start()
    {
        Debug.Log("ピカチュウが登場しました！");
    }

    void Update()
    {
        Debug.Log("ピカチュウが動いています");
    }
}
```
:::

# 問題 3: スクリプトをゲームオブジェクトにアタッチする

1.シーンビューまたはヒエラルキーで、新しいオブジェクト（例: スフィア）を作成します。
2.作成した PikachuController をオブジェクトにアタッチしてください。
・ドラッグ＆ドロップを使うか、インスペクターの「Add Component」を使用します。

# 問題 4: 動作確認

1. Unityエディタで再生ボタンをクリックします。
2. コンソールビューで以下のログが表示されることを確認してください。
・再生開始時: 「ピカチュウが登場しました！」
・再生中: 毎フレーム「ピカチュウが動いています」

# 問題 5: ポケモンのステータスを追加

1. スクリプトに以下の変数を追加してください：
   - `public string pokemonName = "ピカチュウ";`  
   - `public int hp = 100;`  

2. Start関数内のログを以下の形式に変更してください：
   - 「ピカチュウ（HP: 100）が登場しました！」  

3. Update関数内のログを以下の形式に変更してください：
   - 「ピカチュウが現在のHP: 100で動いています！」

:::details ヒント : サンプルコードを見る
```csharp
using UnityEngine;

public class PikachuController : MonoBehaviour
{
    public string pokemonName = "ピカチュウ";
    public int hp = 100;

    void Start()
    {
        Debug.Log($"{pokemonName}（HP: {hp}）が登場しました！");
    }

    void Update()
    {
        Debug.Log($"{pokemonName} が現在のHP: {hp} で動いています！");
    }
}
```
:::

> ログが正しく出力されれば成功です。

# 問題 7: 引数と返り値を使いこなそう！ピカチュウのアクションを拡張

**以下の3種類の関数を作成してください。それぞれの関数には異なる特徴を持たせます。`Start`関数から呼び出して、ログを確認しましょう。**

## 1. 引数なし・返り値なしの関数

**関数名: `Attack()`**  
- ピカチュウが攻撃を行う関数を作成してください。
- 攻撃内容をデバッグログに出力します。

:::details ヒント : サンプルコードを見る
```csharp
void Attack()
{
    Debug.Log($"{pokemonName} は 10万ボルトで攻撃した！");
}
```
:::

## 2. 引数あり・返り値なしの関数

**関数名: `TakeDamage(int damage)`**
-ダメージを引数で受け取り、Hpから差し引いて、残りのHpをデバッグログに出力します。

:::details ヒント : サンプルコードを見る
```csharp
void TakeDamage(int damage)
{
    hp -= damage;
    Debug.Log($"{pokemonName} は {damage} のダメージを受けた！ 残り HP は {hp} です！");
}
```
:::

## 3. 引数あり・返り値ありの関数

**関数名: `Heal(int healAmount)`**  
- ポケモンの回復をシミュレートする関数を作成してください。
- 引数で回復量（`healAmount`）を受け取り、`hp` に加算します。
- 回復後のHPを返り値として返します。
- 回復処理の内容をデバッグログに出力してください。

:::details ヒント : サンプルコードを見る
```csharp
int Heal(int healAmount)
{
    hp += healAmount;
    Debug.Log($"{pokemonName} は 回復して HP が {hp} になった！");
    return hp;
}
```
:::

> ログが正しく出力されれば成功です。

```csharp
using UnityEngine;

public class PokemonActions : MonoBehaviour

    void Start()
    {
        // 引数・返り値なし
        DisplayPokemonName();

        // 引数あり・返り値なし
        TakeDamage(20);

        // 引数あり・返り値あり
        int updatedHP = Heal(30);
        Debug.Log($"最終的な HP: {updatedHP}");
    }

    // 1. 引数・返り値なし
    void DisplayPokemonName()
    {
        Debug.Log($"ポケモンの名前は {pokemonName} です！");
    }

    // 2. 引数あり・返り値なし
    void TakeDamage(int damage)
    {
        hp -= damage;
        Debug.Log($"{pokemonName} は {damage} のダメージを受けた！ 残り HP は {hp} です！");
    }

    // 3. 引数あり・返り値あり
    int Heal(int healAmount)
    {
        hp += healAmount;
        Debug.Log($"{pokemonName} は {healAmount} 回復して HP が {hp} になった！");
        return hp;
    }
}
```
